# 🚀 TravelViet - Laravel React Project

## 🧱 Tech Stack
- Backend: Laravel
- Frontend: React (Vite)
- Database: MySQL
- DevOps: Docker, Docker Compose
- CI/CD: GitHub Actions

---

## ⚙️ Setup project

### 1. Clone project
```bash
git clone https://github.com/yourname/travelviet.git
cd travelviet
```
2. Run Docker
   ```
   docker-compose up -d --build
   ```
4. Access
   ```
   Access
    Frontend: http://localhost:3000
    Backend API: http://localhost:8000
    phpMyAdmin: http://localhost:8081
   ```
